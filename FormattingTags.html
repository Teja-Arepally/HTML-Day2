Formatting Tags

Formatting tags are used to change the appearance of text like making it bold, italic, underlined, small, big, marked, etc.

Bold:Bold Text makes the text bold without giving extra importance

Strong:Highlights taxt as important,brwsers render it in bold

Italic:Makes the text italic without emphasizing importance

Emphasizes:Emphasizes text;browsers render it an italic and screen readers put stress on it

UnderLine:Underlines the text

Mark:Highlight text with a yellow background

Small:Displays text on size smaller than normal

Deleted textShows text that has been deleted/removed; browsers display it with a strike-through.

Inserted text:Shows text that has been added/inserted; browsers display it as underlined.

SubScript:Displays text below the baseline (used in formulas, chemical notations).

Superscript:Displays text above the baseline (used in exponents, math notations).

Big:Thetag is used to increase the font size of text by one level larger than the surrounding text.
